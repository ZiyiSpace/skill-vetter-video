# Phase 1: 内容复杂度评估表

## 工具/Skill 基本信息
- **Skill 名称**：Skill Vetter 🔒
- **来源**：https://llmbase.ai/openclaw/skill-vetter/
- **作者**：spclaudehome
- **核心功能**：AI Agent 技能安全审查工具，安装任何技能前的必备安全卫士

## 核心功能（5个）
1. **来源检查** - 验证技能来源、作者信誉、下载量、更新时间
2. **代码审查** - 检测红旗（curl/wget、外发数据、读取敏感文件、base64解码等）
3. **权限评估** - 分析文件访问、网络访问、命令执行范围
4. **风险分类** - 低/中/高/极高四级风险评级
5. **审查报告** - 生成标准化的安全审查报告

## 适用场景（4个）
1. 安装 ClawHub 技能前审查
2. 安装 GitHub 开源技能前审查
3. 评估他人分享的技能
4. 企业内部技能安全合规

## 配置复杂度

## 版本推荐

**推荐版本：STANDARD（10场景）**

**推荐理由：**
1. 功能较多（5个核心功能），需要足够篇幅展示
2. 适用场景丰富（4个），值得详细说明
3. 安全审查是高频需求，10场景版本性价比最高
4. 内容足够支撑 45-60 秒视频，适合短视频平台

**其他版本适用情况：**
- **COMPACT（6场景）**：如果只需要快速介绍核心功能
- **EXTENDED（12场景）**：如果需要详细演示审查流程
