## Narrative Framework (Updated)

### 5-Segment Structure (Based on Gemini Analysis)
| Segment | Duration | Purpose | Core Strategy |
|---------|----------|---------|---------------|
| **1. Hook** | 3-5s | Grab attention | Extreme statement + Conflict + Crisis |
| **2. Proof** | 10-15s | Build trust | Data + Authority + Scale |
| **3. Breakdown** | 30-40s | Show value | 3 layers: Beginner → Advanced → Expert |
| **4. Resonance** | 10-15s | Hit pain points | "Thoughtfulness" + Remove barriers + Real experience |
| **5. CTA** | 5-10s | Drive action | Save + Urgency + Emotional close |

### Hook Formulas
- **断财路型**: "This will make XX lose their jobs"
- **降维打击型**: "Stop paying for XX, this free tool is better"
- **效率焦虑型**: "If you don't know this, you're wasting X hours daily"
- **极端陈述型**: "The strongest XX tool on GitHub, bar none"

### Script Structure Requirements
- First sentence must be spoken within 1 second
- Remove all filler words ("大家好", "今天我想分享")
- Every visual change every 1-2 seconds
- Emphasize "Save" over "Like" (save rate = long-term value signal)
- Include screen recording guidance (red boxes, yellow arrows)

## Douyin Platform Compliance

### Legal Framework (China)
1. **《互联网信息服务深度合成管理规定》** (2023-01-10)
   - AI-generated content must be labeled
   - Unauthorized face swaps prohibited
   - Violations: fines, potential criminal liability

2. **《个人信息保护法》**
   - Consent required for biometric data processing
   - Separate consent for sensitive personal information

### Douyin Platform Rules
1. **AI Content Labeling** (2025 new regulation)
   - Must check "Video generated by AI" option
   - Recommended on-screen text: "AI Technology Demo"

2. **Deepfake Red Lines**
   - ❌ Celebrities, politicians, influencers
   - ❌ Adult/sexual content
   - ❌ Misleading content (fake as real)
   - ✅ Self portrait, official samples, royalty-free images

3. **External Traffic Restrictions**
   - ❌ "GitHub", "link", "click download", "DM me"
   - ✅ "G站", "check pinned", "surprise in profile"

4. **Sensitive Words**
   - ❌ "白嫖" → ✅ "免费获取" / "开源学习"
   - ❌ "被和谐" → ✅ "下架" / "失效"
   - ❌ "换脸" → ✅ "形象定制" / "面部编辑" / "视觉合成"
   - ❌ "让XX失业" → ✅ "提升效率" / "行业变革"
